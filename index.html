Recent Updates

v1.2.2
2026-05-24
  • Fix: Draft tab no longer flickers through historical drafts on each auto-scan
  • Fix: viewing an earlier pick in the current pack no longer gets interrupted by new picks being scanned — only advances automatically when already on the live pick, on an older pack, or when a new draft starts
v1.2.1
2026-05-24
  • Fix: Contender Draft matches incorrectly labeled as Premier Draft when opponent was in a different event type
  • Fix: converted mana cost for double-faced cards now only considers the front face
  • History: Premier Draft and Contender Draft are now separate draft run groups within the same set view
  • History: draft runs sorted by when the draft started, not when the last game was played
  • History tab moved above Statistics in the sidebar
  • Statistics: card stats for a draft set now merges Premier and Contender Draft data together
v1.2.0
2026-05-23
  • History: draft runs are now collapsible — latest draft expanded by default, click the record to toggle
  • History: collapsed drafts show record, format, color pips, date, and a delete button
  • History: draft tier color coding (brown → black → silver → gold → red for 0–7 wins)
  • History: progressive combo column disclosure in the format selector — more detail as the panel widens
  • Statistics: Premier Draft and Contender Draft merged into a single Draft row per set
v1.1.0 – v1.1.3
2026-04 – 2026-05
  • Contender Draft recognized as a supported format (alias of Premier Draft)
  • Fix: 17Lands lookup for double-faced Strixhaven cards
  • Fix: export/import now correctly includes card stats and draft data
  • Data Management UI improvements and import flow clarification
v1.0.0
2026-03
  • Standalone Windows installer — no Node.js required
  • Draft Assistant: live pack ratings from 17Lands CSV data
  • Deck Builder: color pool analysis and hypergeometric calculator
  • Match History: format cards with color combo stats and win rates
  • UX overhaul: deck view, format detection, and color classification
  • Automatic match tracking, inventory updates, and system tray integration
